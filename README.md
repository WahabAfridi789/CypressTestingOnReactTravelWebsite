This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

Once you clone this repository, you can run the following command in the `CypressTestingOnReactTravelWebsite` directory.

### `npm install`

Then run the following command to run the React project.

### `npm start`

 Once the React project runs successfully on localhost, type the following commands:

### `cd testing` -> testing directory

Then again run the following command:

### `npm install`

Finally, run the following command:

### `npx cypress open`

And there you are, good to go! Enjoy!!
